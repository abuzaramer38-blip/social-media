# Social Manager Pro

Apni shop ke social media accounts ek jagah manage karo.

## Features
- TikTok, Instagram, Facebook, YouTube account tracking
- Shop info, bio, address, phone save karo
- Posts add karo aur sab platforms pe track karo
- Activity history
- Password manager

## Deploy on Vercel

1. GitHub pe upload karo (niche steps)
2. vercel.com pe jao → "New Project"
3. GitHub repo select karo
4. Deploy!

## GitHub Upload Steps

```bash
git init
git add .
git commit -m "Social Manager Pro - Initial"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/social-manager.git
git push -u origin main
```

## Local Run

```bash
npm install
npm run dev
```
